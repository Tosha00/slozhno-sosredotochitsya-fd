## Описание проекта  
Перед вами проектная работа третьего спринта учебного курса от Яндекс Практикума. В отличие от предыдущих работ, здесь основной упор сделан на адаптивность и построение гибкого интерфейса, который корректно отображается на различных устройствах и разрешениях экрана.

Проект развивает навыки работы с уже существующей структурой, но требует более глубокого понимания поведения интерфейса: элементы больше не имеют фиксированных размеров, а подстраиваются под условия среды.

---

## Используемые технологии  
В проектной работе активно применяются современные возможности CSS для создания адаптивной и масштабируемой вёрстки:

- **Адаптивная вёрстка** — интерфейс корректно работает на мобильных, планшетах и десктопах  
- **Относительные единицы измерения** (`rem`, `vw`, `%`) — для гибкости и масштабируемости  
- **CSS Grid Layout** — построение сложных сеток и управление расположением элементов  
- **Media Queries** — использование брейкпоинтов для изменения структуры и стилей  
- **CSS-переменные** — реализация двух цветовых схем (светлой и тёмной темы)  
- **Гибкая типографика** — использование `clamp()` для адаптивных размеров текста  

---

## Особенности проекта  
- Реализация **двух тем оформления** с помощью CSS-переменных  
- Переход от фиксированной к **адаптивной логике построения интерфейса**  
- Использование **гридов и их возможностей** для построения layout  
- Повышенное внимание к **поведению элементов при изменении размеров экрана**  
- Формирование системного подхода к вёрстке, а не точечной настройки стилей
